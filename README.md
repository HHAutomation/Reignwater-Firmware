# Reignwater Controller Firmware v0.8.5

**Release Date:** November 29, 2025

## Installation

### Via App (Recommended)
1. Open Reignwater Controller app
2. Go to Settings → Check for Updates
3. Configure WiFi (Settings → WiFi Configuration)
4. Tap "Install Update"

### First-Time Setup
1. Connect to device via Bluetooth (BLE)
2. Default PIN is 7777 (App automatically connects)
3. Change PIN in Settings for security (Automatic process)
4. Configure WiFi for wireless updates (WiFi is only used for firmware updates)

## Requirements

- **Mobile App**: Reignwater Controller v0.8.0 or higher
- **WiFi**: Required for updates only.
